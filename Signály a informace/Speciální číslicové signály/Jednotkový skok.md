**Jednotkový skok** $u[n]$ je druh signálu, který je definován jako
$$\Large
u[n] = \begin{cases}
	0 & \text{pro } n \lt 0 \\
\\
	1 & \text{pro } n \ge 0
\end{cases}
$$

Jedná se o diskrétní obdobu [[Heavisideova funkce|Heavisideovy funkce]].
