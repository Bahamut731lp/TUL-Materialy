# Veřejné konstrukce
Veřejné konstrukce jsou takové, které ve své deklaraci používají klíčové slovo `public`.

```java
//Veřejná třída
public class Trida {
	//Veřejný atribut
	public boolean otevreno = true;
	//Veřejná metoda
	public static void main(String[] args) {
		...
	}
}
```
