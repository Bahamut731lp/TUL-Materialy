# Vzorečky pro integraci
Jedná se v podstatě o vzorečky pro [[Pravidla pro práci s derivacemi|derivacemi]], akorát se čtou naopak

$$\Large\begin{aligned}
\int{0dx} &= C \\ 
\int{1dx} &= x + C \\
\int{x^n}dx &= \frac{x^{n+1}}{n+1} + C \\
\int{\frac{1}{x}}dx &= \ln{|x|} + C \\
\int{a^x}dx &= \frac{a^{x}}{\ln{a}} + C \\
\int{e^x}dx &= e^x + C \\
\end{aligned}$$
